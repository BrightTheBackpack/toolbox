# Hack Pack

The Hack Pack is GitHub’s Student Developer Pack, given instantly & for free to all Hack Clubbers. When you register, you can get a pack of Hack Club stickers mailed to you!

## How to redeem

1. Go to [hack.af/pack](https://hack.af/pack)
2. Sign in with your GitHub account
3. List what Hack Club/hackathon you’re a part of
4. Add your address if you want stickers (optional)
5. Click Submit & you’ll be redirected to GitHub
6. Profit! You’ll know you’ve got it when GitHub emails you confirmation.
